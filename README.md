 My-App  
Maven-Based Java Application with Jenkins CI/CD Pipeline

This repository demonstrates a basic DevOps CI/CD workflow using a Java Maven application integrated with Jenkins.  
It is designed for DevOps internship / beginner-level practice to understand build automation and pipeline concepts.

Project Overview

- Java application built using Maven
- CI/CD pipeline configured using Jenkins
- Source code hosted on GitHub
- Pipeline auto-triggered using SCM Polling
- Focus on DevOps fundamentals: build, test, and automation

Tech Stack

- Language: Java  
- Build Tool: Maven  
- CI/CD: Jenkins  
- Version Control: Git & GitHub  
- Operating System: Linux  



 CI/CD Pipeline Flow

1. Developer pushes code to GitHub repository  
2. Jenkins detects changes using SCM polling  
3. Jenkins pulls latest code  
4. Maven build is triggered  
5. Build status is displayed in Jenkins dashboard
